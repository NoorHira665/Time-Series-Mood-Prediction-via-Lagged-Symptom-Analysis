# Time-Series-Mood-Prediction-via-Lagged-Symptom-Analysis
Forecasts daily moodswing severity (0-5 scale) for women using lagged (1, 3, 7-day) self-reported symptoms, sleep and cycle data. Features group-wise imputation and Ridge Regression for interpretable results.

Dataset used: https://www.physionet.org/content/mcphases/1.0.0/
